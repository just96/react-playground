# React Movies App

A simple **React** application to manage movies, with the ability to **add, edit, and remove** movies dynamically.

---

## Features

- Fetch and display movies from a simulated API.  
- Add new movies to the list.  
- Edit existing movie titles.  
- Remove movies from the list.  
- Generic and reusable components (`Input`, `Button`) and specific components (`MovieForm`, `MovieList`) for better structure.

---

## Technologies

- **Frontend:** React, JSX, TailwindCSS, daisyUI
- **Backend/API:** Simulated data via fetch (JSON fake API)  
- **State management:** useState and useEffect  
- **Componentization:** Generic and reusable components
